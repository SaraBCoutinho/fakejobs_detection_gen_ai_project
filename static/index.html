Como a IA participa

O agente usa o Ollama para gerar uma explicação estruturada. O score continua sendo calculado por uma ferramenta determinística: o modelo explica as evidências, mas não pode alterar o veredito.

  1. Prompt protegido: a vaga é delimitada como dado não confiável para reduzir prompt injection.
  2. Tool calling: o agente precisa chamar calcular_risco_vaga antes de responder.
  3. Saída estruturada: resumo, recomendação, verificações, limitações e alerta de privacidade.
  4. Fallback: se o Ollama falhar, a análise determinística continua disponível e a interface informa isso.
